ConvertObjectToDocument(Object $object, string $rootElementName) : \DOMDocument
static

Converts an object to a DOMDocument.

The root element name is passed in as a parameter, and each field of the object becomes a child element. Array values are represented by multiples instances of that element. Methods on the object are ignored. There is no support for XML attributes.
